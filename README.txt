Custom post type for cook recipes

== Description ==

You can post cook recipes easily with this post but i will create new views for this plugin in next version.

== Usage ==

When you activate the plugin, you will see add new recipe link at admin panel.

== Installation ==

1. Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation
2. Activate the Plugin from Plugins page
